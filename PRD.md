# Cafe Management & Landing Page Website PRD

## 1. Product Overview

A modern web platform for a local Egyptian cafe that showcases the brand, menu, gallery, and allows customers to make reservations. The system also includes an admin dashboard for managing products, reservations, gallery images, and customer messages.

---

## 2. Product Goals

### Business Goals

* Increase cafe visibility online.
* Allow customers to browse the menu easily.
* Enable online table reservations.
* Provide a modern digital experience.
* Simplify cafe content management.

### User Goals

* View menu items and prices.
* Explore the cafe atmosphere through images.
* Contact the cafe easily.
* Reserve a table online.

---

## 3. Target Users

### Customer

Visitors interested in viewing the menu, exploring the cafe, and making reservations.

### Admin

Cafe owner or manager responsible for maintaining content and reviewing reservations.

## 5. Functional Requirements

### Reservation System

Customer must be able to:

* Enter name
* Enter phone number
* Select reservation date
* Select number of guests

System must:

* Save reservation
* Mark status as Pending

### Product Management

Admin must:

* Add menu items
* Upload product image
* Assign category
* Set price

### Contact Form

Customer must:

* Submit inquiry

System must:

* Store inquiry in database
